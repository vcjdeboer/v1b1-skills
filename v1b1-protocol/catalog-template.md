## Substitution rules

- `<placeholder>` tokens are replaced wholesale.
- Frontmatter fields conditional on mode (`intent_text`, `confirmation_answers`) are omitted when not applicable, NOT rendered as empty strings.
- `(or void / None ...)` and `(if empty, render ...)` annotations are NOT preserved in the rendered catalog — they are template guidance only.
- Per-entry blocks repeat in deterministic order (FR-009 / VR-6).
- `entry_count` frontmatter field MUST equal the number of `## ` subsections rendered under `# Commands` (audit step 3 verifies).
